## ADDED Requirements
### Requirement: Compass Feedback
The system MUST provide visual and haptic feedback when the user is aligned with the Qibla.

#### Scenario: Alignment Persistence
- **WHEN** the device is aligned with Qibla
- **THEN** the "You're facing Qibla" message MUST remain visible as long as alignment is maintained
- **AND** the visual pulse indicator MUST be active only during alignment

### Requirement: Visual Aesthetics
The system MUST use premium typography.

#### Scenario: Custom Fonts
- **The system** SHALL use a custom font family (e.g., Inter) for all text elements to ensure a premium feel.
