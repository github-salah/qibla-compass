## MODIFIED Requirements
### Requirement: Compass Visualization
The system SHALL display a compass interface that visualizes the user's current heading and the direction of the Qibla.

#### Scenario: Modern Layout
- **WHEN** the compass is displayed
- **THEN** the cardinal directions (N, S, E, W) MUST be positioned on the outer ring
- **AND** the Qibla indicator MUST be clearly visible without obstructing the view
- **AND** the direction arrow MUST use a modern, sleek design
